## Goal

Produce a ranked idea list that reflects strategic value, evidence quality, and implementation efficiency.

## How

Sort the shaped ideas in this order:

1. strategic alignment
2. evidence strength
3. competitor gap
4. dependency count
5. scope efficiency

If two ideas are close, prefer the one with fewer dependencies and stronger evidence.

## Done-when

The shaped idea set is ordered from strongest to weakest with a clear rationale for the ranking.
